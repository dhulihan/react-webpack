# react-webpack

A sandbox for react/webpack. 

## Install

```sh
npm install
```

## Run

```sh
grunt serve # dev server: localhost:8000/webpack-dev-server/
grunt serve dist # dist server localhost:8000/
```

## Buid

```sh
grunt build
```

## Notes

* Files 
	* `webpack.dist.config.js` - Build/dist config
